# Future Wear

![Project Banner](https://github.com/user-attachments/assets/0aa67016-6eaf-458a-adb2-6e31a0763ed6)

A modern, brutalist-inspired e-commerce landing page for a futuristic apparel and tech-wear brand. Built with cutting-edge web technologies to deliver a highly interactive and visually striking user experience.

## ✨ Features

- **Brutalist Design System:** Custom UI featuring hard shadows, high contrast, and striking typography (`font-display`).
- **Dynamic Animations:** Immersive scroll animations, hover effects, parallax, and continuous marquees powered by `motion/react`.
- **Responsive Layout:** A fluid grid system and responsive design ensure the store looks great on any device.
- **Section Components:**
  - `Hero`: High-impact landing area with glitch text and scroll-linked rotation/parallax effects.
  - `ProductGrid`: E-commerce showcase featuring stagger animations, hover state transitions, and product categories.
  - `Testimonials`: Customer reviews section built with the same striking design language.
  - `MoreInfo` & `Footer`: Supplementary content and navigation components.

## 🛠 Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (v15)
- **UI Library:** [React](https://react.dev/) (v19)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (v4)
- **Animations:** [Motion](https://motion.dev/) (`motion/react`)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** TypeScript

## 🚀 Getting Started

Follow these instructions to run the project locally.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository** (or download the source code):
   ```bash
   git clone <repository-url>
   cd remix_-neo-market-2026-v2
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```
   *or if you use yarn/pnpm:*
   ```bash
   yarn install
   # or
   pnpm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open the app:**
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `/app`: Next.js App Router pages and global layouts.
- `/components`: Reusable React components (`Hero`, `ProductGrid`, `Testimonials`, etc.).
- `/public`: Static assets like images used in the application.
